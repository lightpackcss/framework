# List Component

**Classes:** `.list`

**Features:**
- Styled lists with spacing and improved typography

**Example**
<div class="demo-container">
  <ul class="list">
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
  </ul>
</div>

**Usage:**
```html
<ul class="list">
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>
```

**Customization:**
- Add utility classes for custom spacing or color
